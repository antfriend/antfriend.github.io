Toot Toot Engineering (Runner)

This repo includes a local agent runner for the Toot Toot Engineering workflow.

Quick start

  • Install dependencies: pip install -r requirements.txt or pip install openai
  • Set API key: export OPENAI_API_KEY="..." (macOS/Linux) or setx OPENAI_API_KEY "..." (Windows PowerShell)
  • Run agent: python tte_agent.py (or python3 tte_agent.py)
  • Run monitor: ./run_tte.sh, run_tte.bat, or .\run_tte.ps1

Notes

  • Workflow instructions live in the GitHub repo.
  • The agent reads and operates inside the repo workspace.
  • tte_monitor.py uses standard library, but needs tkinter.

Install tkinter

  • macOS: brew install python-tk
  • Ubuntu/Debian: sudo apt-get install python3-tk
  • Fedora: sudo dnf install python3-tkinter
  • Arch: sudo pacman -S tk
  • Windows: python.org installer with "tcl/tk and IDLE" enabled

Troubleshooting

  • "Missing OPENAI_API_KEY": confirm the env var in your shell.
  • Cannot import openai: reinstall deps or activate the right venv.
